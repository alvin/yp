Screen Notes — ‘Date Search’ Return Screen

This should be a simple list of all the guests, sorted alphabetically by last name, that have an ‘arrival date’ that matches the date searched. When a particular name is selected, the appropriate Transaction Screen is returned.

Feature file: date_search_results.feature

Related screen: Transaction Screen

Scope note: the written requirements also say date search can return matching check-in or check-out dates; that difference is preserved in the feature file and main scope docs.
